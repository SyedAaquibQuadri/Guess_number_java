# About the program

A simple command-line game written in Java where the player tries to guess a randomly generated number within a limited number of attempts.

## About

This is a biginner Java mini project that demonsrates core programming concepts like input handling, loops, conditional statement, random number generation. The computer picks a secret number and gives you hints after each guess -  high or  low - until you guess the right number.

if you exhausted the limit (which is 5) the computer asks you if you want to continue or not, if the user types yes the program runs afterwards.
